# Word Guessing Game

A simple word guessing game developed in Python. The player chooses a difficulty level and tries to guess a randomly selected programming-related word before running out of chances.

## Features

* Three difficulty levels:

  * Easy (7 chances)
  * Normal (5 chances)
  * Hard (3 chances)
* Random word selection
* Win and loss tracking
* Game statistics displayed at the end

## Technologies

* Python 3
* `random` module

## How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/word-guessing-game.git
   ```

2. Open the project folder:

   ```bash
   cd word-guessing-game
   ```

3. Run the program:

   ```bash
   python main.py
   ```
